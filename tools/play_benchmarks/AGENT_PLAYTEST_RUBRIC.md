# Agent Playtest Rubric

For every story, each playtest agent should report:

1. ending and turn count
2. narration coherence
3. suggested action coherence
4. state-bar / stance-bar credibility
5. protagonist identity clarity
6. one-sentence overall player-feel verdict

Required flags to extract from qualitative feedback:

- `flat_state_feedback`
- `player_identity_confusion`
- `templated_opening`

Preferred wording:

- concise English
- 4 short numbered points
- one strongest issue only
